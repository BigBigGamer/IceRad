sf = shapefile.Reader('') - читает файл .shp и все с ним связанные

sf.shapes() - возвращает список шейп-объектов

    Каждый шейп имеет атрибуты
        .shapeType - Int, тип шейпа
        .shapeTypeName - Char, имя типа шейпа
        .bbox - list, координаты ограничивающего прямоугольника
        .parts - list, содержит индексы начала частей шейпа. если 0, то часть одна
        .points - list , содержит пары (tuples) (x,y) - координаты каждой точки шейпов


sf.fields - возвращает список с именем атрибута, тип атрибута, размер  и еще что-то (для всех шейпов)

records = sf.records() - возвращает список записей

record = records[i] - выбрать конкретную запись

a = record['NAME'] - способ достать из записи конекретный атрибут
a = record.NAME - способ достать из записи конекретный атрибут
a = record[5] - способ достать из записи конекретный атрибут
